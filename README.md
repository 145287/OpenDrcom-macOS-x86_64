# OpenAutoDrcom-macOS-x86_64（哆点自动拨号 macOS 64位版）
### 简介：此程序是Drcom的第三方自动拨号器，主要用于解决自动重连的问题。

#### 使用说明
- macOS 10.12或更高版本开启“任何来源”后才可以运行本程序（因为我没注册苹果付费帐号，一年688大洋，穷）
- 在“终端”中执行以下指令，执行后输入管理员密码即可。

~~~
sudo spctl --master-disable
~~~
